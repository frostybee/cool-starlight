# Slide Viewer Component

The **Slide Viewer** is a powerful Astro component that automatically converts your page content into an interactive slide presentation system. It provides a rich set of features for navigation, customization, and content interaction.

---

## Core Presentation Features

### Automatic Slide Generation
The component intelligently parses your page content and creates slides based on:

1. **Heading-based divisions** - Each major heading starts a new slide
2. **Content sections** - Logical content breaks are preserved
3. **Code blocks and lists** - Special formatting is maintained
4. **Image and media support** - Visual content is properly displayed

### Navigation System
Navigate through your presentation using multiple methods:

- **Keyboard shortcuts**: Arrow keys, Page Up/Down, Home/End
- **Navigation buttons**: Previous/Next slide controls
- **Direct slide access**: Go to any slide number instantly
- **Mobile gestures**: Touch-friendly mobile navigation
- **Progress tracking**: Visual progress bar shows presentation completion

### Viewing Modes
The component offers three distinct viewing experiences:

1. **Slide Mode** - Traditional presentation view with one slide at a time
2. **Reading Mode** - Continuous document view for in-depth reading
3. **Overview Mode** - Thumbnail grid showing all slides at once

## Advanced Search & Discovery

### Real-time Search
- **Instant search** across all slide content as you type
- **Highlighted matches** with search term emphasis
- **Filtered results** showing only relevant slides
- **Search persistence** across different viewing modes

### Search Modal
Access the advanced search interface with enhanced features:

1. **Debounced search** for smooth performance
2. **Keyboard navigation** through search results
3. **Content snippets** showing search context
4. **Quick navigation** to matching slides

### Interactive Table of Contents
- **Smart TOC generation** based on content structure
- **Slide previews** on hover for quick content overview
- **Reading time estimates** for each slide section
- **Bookmark indicators** showing saved slides
- **Collapsible sidebar** to maximize content space

## Interactive Tools & Features

### Laser Pointer System
A sophisticated annotation tool with multiple drawing modes:

- **Freehand drawing** for natural sketching and highlighting
- **Shape tools**: Circles, rectangles, and straight lines
- **Customizable settings**:
  1. Adjustable colors and thickness
  2. Configurable fade duration (0.5-5 seconds)
  3. Reverse fade animation options
  4. Live preview of settings changes

### Bookmark Management
Keep track of important slides with the bookmark system:

- **Quick bookmarking** with keyboard shortcuts (B key)
- **Bookmark panel** showing all saved slides with previews
- **Persistent storage** - bookmarks saved across sessions
- **Visual indicators** in thumbnails and TOC
- **Easy navigation** to bookmarked content

### Font Size Control
Accessibility-focused text scaling:

- **Dynamic font adjustment** with + and - controls
- **Range**: 60% to 500% scaling
- **Reset functionality** via double-click
- **Persistent preferences** saved to local storage
- **Mobile-optimized** controls for touch devices

## Accessibility & Keyboard Support

### Comprehensive Keyboard Shortcuts
Master these shortcuts for efficient navigation:

#### Navigation
- `←` `→` - Previous/Next slide
- `Page Up` `Page Down` - Previous/Next slide  
- `Home` - Go to first slide
- `End` - Go to last slide
- `G` - Go to specific slide number

#### Search & Tools
- `/` or `F` - Open search modal
- `L` - Toggle laser pointer
- `S` - Open laser pointer settings
- `T` - Cycle through themes
- `B` - Bookmark current slide
- `Shift + B` - Toggle bookmarks panel

#### General
- `Esc` - Close slideshow or modal
- `Shift + ?` - Show keyboard help

### Accessibility Features
- **ARIA live regions** for screen reader announcements
- **Keyboard focus management** with proper tab order
- **Screen reader optimized** labels and descriptions
- **High contrast support** across all themes
- **Semantic HTML structure** for assistive technologies

## Technical Implementation

### Modular Architecture
The component is built with a modular system of specialized managers:

- **SlideViewer.js** - Core presentation engine
- **SearchManager.js** - Advanced search functionality
- **LaserPointerManager.js** - Drawing and annotation tools
- **BookmarkManager.js** - Slide bookmarking system
- **ThemeManager.js** - Theme switching and compatibility
- **TOCManager.js** - Table of contents generation
- **KeyboardHelpManager.js** - Keyboard shortcut system
- **FontManager.js** - Font size adjustment
- **FullscreenManager.js** - Fullscreen presentation mode
- **ReadingModeManager.js** - Alternative reading view
- **ThumbnailManager.js** - Slide overview generation
- **MobileMenuManager.js** - Mobile interface optimization

### Performance Optimizations
- **Lazy content loading** for smooth performance
- **Debounced search** to prevent excessive operations
- **Throttled scroll listeners** for efficient updates
- **Local storage caching** for user preferences
- **Efficient DOM manipulation** with minimal reflows

### Browser Compatibility
- **Modern ES6+ JavaScript** with module support
- **CSS Grid and Flexbox** for responsive layouts
- **Local Storage API** for preference persistence
- **Canvas API** for laser pointer functionality
- **Fullscreen API** for immersive presentations


### Content Optimization
Structure your content for optimal slide generation:

1. **Use clear headings** to define slide boundaries
2. **Keep sections focused** - one main concept per slide  
3. **Include visual elements** like code blocks and images
4. **Write concise content** appropriate for presentation format

### Best Practices
- **Test responsive behavior** on different screen sizes
- **Verify keyboard navigation** works smoothly
- **Check theme compatibility** with your site design
- **Optimize content length** for slide format presentation

The Slide Viewer transforms any content page into a professional presentation experience with minimal setup while providing powerful customization options for advanced users.
